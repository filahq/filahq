# FilaHQ OG Image Design

**Date:** 2026-04-11

## Goal

Create a text-first Open Graph image for Statify that matches the landing page tone and uses the approved headline:

`Take Filament stats beyond the panel.`

## Format

- Size: `1200x630`
- Output file: `public/og-statify.png`

## Composition

- Small label near the top:
  `Statify by FilaHQ`
- Large dominant headline:
  `Take Filament stats beyond the panel.`
- Optional small supporting line near the bottom:
  `Filament plugins and integrations for cross-platform stats`

## Visual Direction

- Dark graphite background using `#343434`
- Yellow signal accent using `#ffcc00`
- Minimal background signal lines or texture only
- Strong left alignment
- Generous safe margins so the headline survives social crop and compression
- No dashboard UI, screenshots, or dense illustration

## Intent

The image should feel like a distilled poster version of the landing page hero: direct, legible, technical, and product-focused.
